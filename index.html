Database for Beginners

What is a Database?

A database is a collection of related data. Most often when we speak of a database, we refer to an electronic database. An electronic database is a database that runs on a computer or a server. For the tutorials on this website, we will mainly focus on electronic databases precisely - relational databases. A database can also refer to a non-electronics database. For example, a phone-book, library card catalog, a dictionary, etc..

Database Management Systems (DBMS)

A database management system is a system software used for creating and maintaining the structure of a database. It is used to input, manipulate, and retrieve data.

Types of Database Management Systems (DBMS)

  • Hierarchical Database Management Systems - organized like a tree like structure. Often used in large institution such as banks and insurance companies.
  • Network Database Management Systems - use a network structure to create relationships between entities. Often used on larger digital computers.
  • Relational Database Management Systems - is a system used for managing relational databases.
  • Object-oriented Database Management Systems - supports the creation and modeling of data as objects.
Note: On this website, our primary focus is Relational Database Management Systems.

Relational Database Management Systems (RDBMS)

A relaional database management system is a DBMS that is based on the relational model introduced by Edgar Frank Codd. RDMBS needs to allow the user create one or more databases objects associated with that database such as tables, relationships, views, and queries. In 1974, IBM began developing System R, a research project to develop prototype RDBMS. The first commercially available RDBMS was Oracle, released by Relational Software, today it is known as Oracle Cooperation. Relational database is the basic for SQL and all modern databases. Common RDBMS are Oracle, SQL Server, DB2, MySQL, PostGres SQL, Access, etc.. Screenshots or examples shown on this website will be Oracle SQL developer.

What is SQL?

SQL (Structured Query Language) is the sublanguage with commands used to create database objects and manipulate data stored in a database. It is not considered a programming language like Java, C++, or C#.

Basic Database Terminology

  • Character - is the basic unit of data, and it can be a letter, number, or special symbol.
  • Field - a group of related characters.
  • Record - a collection of fields.
  • File - a group of records about the same types of entities.
  • Column - a field is commonly refered to as column in the physical design when creating a database.
  • Row - a record is also called a row.
  • Table - a file is known as a row.
  • Entity - an object of concern to a database. It is used in the logical design phase.
  • Attribute - describes or define the aspect of a database entity. It is used in the logical phase but also know as a column in the physical design.

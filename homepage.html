Database for Beginners

What is a Database?

A database is a collection of related data. Most often when we speak of a database, we refer to an electronic database. An electronic database is a database that runs on a computer or a server. For the tutorials on this website, we will mainly focus on electronic databases precisely - relational databases. A database can also refer to a non-electronics database. For example, a phone-book, library card catalog, a dictionary, etc..

Relational Databases

Relational database management system is a database management. Relational database is the basic for SQL and all modern databases. SQL is the language relational databases used to create their objects and to modify and retrieve data.

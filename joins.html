A database is a collection of related data. Most often when we speak of a database, we refer to an electronic database. An electronic database is a database that runs on a computer or a server. For the tutorials on this website, we will mainly focus on electronic databases precisely - relational databases. A database can also refer to a non-electronice database. For example, a phonebook, library card catalog, a dictionary, etc..
